Calculate


This is an interactive website, where the user can input various types of dates (between the years 1900 to 2100), and output a table of values for each planet. Pluto is not included in the calculations, as it is extremely far away from the sun, and has a negligible mass compared to the rest of the solar system bodies. The system is in a heliocentric equatorial frame.

The different input choices are listed below:

  1. Julian Date – this is in Universal Time (UT1), where time is independent of station location (Green, 1988:245)2. Use of decimals is permitted in this input.
  2. Day of year – this is a number (1-366.25) that represents the day of the year, where the decimal portion is converted to hours, minutes, and seconds.
  3. Date – this is a standard date (MM/DD/YYYY) and it assumes that the hour is noon, minutes and seconds are zeroed.
  4. Date-time – this is a standard input date and time (MM/DD/YYYY/hh/mm/ss) where the user can input any date and time of day.

Each input is converted to JDUT1, if needed, and then run through the Meeus algorithm to compute the results for each planet. The converted Julian Date is provided above the table for convenience.

There is also a figure provided, which shows the relative position of the planets with respect to each other, not to scale. The orbit is drawn circular, but the eccentricity of each planet at that time is documented in the table. The relative radius of each planet is not to scale either; the planets all have a fixed radius in the figure for easier viewing of the graphic. The angle of the planets, relative to the sun and other planets, is accurately depicted.
