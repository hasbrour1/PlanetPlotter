Conclusion


Obtaining the position and velocity of the planets at a given point in time can be achieved with relative ease. One of the more common approaches is using the Meeus algorithms for preliminary estimates. Using a third order polynomial, Meeus equations find the orbital elements of each planet, which yield position and velocity. The JPL DE405 is a more accurate method of calculating position of the planets. DE405 series uses numerical integration to propagate the planets forward in time. Based on the error analysis conducted between the two different methods, the Meeus algorithm is a great starting point when planning a mission, though a higher fidelity model is required for later phases. The Meeus equations are relatively simple to understand and the output values are comparable to the advanced models. The found values are sufficiently accurate enough to be used for initial estimates.
